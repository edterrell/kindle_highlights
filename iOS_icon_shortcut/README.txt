# 📚 Highlights Shortcut Setup

This bundle includes:
- `Remember Highlights.shortcut` — a Siri-enabled shortcut that opens your Kindle highlight uploader.
- `highlights-icon.png` — a custom icon (arrow into brain) for your Home Screen.
- `README.txt` — this instruction file.

---

## ✅ How to Install the Shortcut

### On iPhone or iPad:

1. Open the **Shortcuts** app.
2. Tap the `+` to create a new shortcut 
3. Rename it to **Remember Highlights** (if needed).
4. Test with Siri: “Hey Siri, Remember Highlights”.

---

## 🧠 Add to Home Screen with Custom Icon

1. In the Shortcuts app, tap `⋯` on the `Remember Highlights` shortcut.
2. Tap the **Share icon** → **Add to Home Screen**.
3. Tap the default picture icon → Choose `highlights-icon.png` from Files.
4. Tap **Add**. Done!
